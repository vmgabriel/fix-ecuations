menu(self)

source code 

Metodo del menu principal, enfocado en mostrar un menu para quedar el usuario seleccione con la entrada, en este caso el menu muestra las siguientes opciones:

   1. Ingresar Ecuaciones: Enfocado en resolver ecuaciones I{X} por I{X}
   2. Evaluar arbol: Enfocado en resolver un arbol previamente guardado
   3. Imprimir arbol PosOrden: Imprime en metodo posOrden
   4. Salir: Da fin al programa.
Returns: int
valor numero de seleccion del usuario
