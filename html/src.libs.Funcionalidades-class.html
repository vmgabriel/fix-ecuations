despejar(self, arbol, letra)

source code 

Metodo de alta funcionalidad recursivo y que posee una cantidad de herramientas internas su funcionalidad es complicada ya que se implemento como un algoritmo de fuerza bruta para cualquier tipo de arbol que tenga la herramienta:

             (=)      Resuelve Despejando A
         (+)     (*)                  ____(=)____
       (A) (3) (5) (6)               (A)        (-)
                                             (*)   (3)
                                           (5) (6)
       En este caso siendo A=(5*6)-3

Notese la forma en la que se resuelve

Parameters:
  • arbol (Nodo()) - Arbol que se va a desejar, se va a alterar
  • letra (str) - Letra que se va a despejar, ya que pueden que hayan varias
Returns: Nodo()
Arbol ya despejado
